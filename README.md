## 📝 Summary
책을 읽는 사람들이 줄어들고 있다. 책을 읽고 책갈피를 사용하여 표시할 수 있고 그날의 감정과 책 갈피마다 요약을 통해서 나만의 책을 만들고 다른 사람들에게 공유하며 좋아요 기능을 
통해서 보람을 느낄 수 있어 많은 사람들이 책을 읽도록 기여할 수 있다.
